# 5-Day Structured Plan

* What is Cloud Computing?
* Types: IaaS, PaaS, SaaS
* Deployment models (Public/Private/Hybrid)
* AWS Global Infrastructure (Regions, AZs, Edge)
* AWS Services
* S3 (object storage)

### Hands-on
* Create AWS Free Tier account
* Explore AWS Console
* Setup Billing Alert 💰
* Create S3 bucket + upload files

### Tip
You should **log in + click + explore** on Day 1 itself.
