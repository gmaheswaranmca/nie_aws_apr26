Steps
1. Search in AWS Console 
    "IAM" -> Select and Go Inside 
    We are in IAM Work Area.

    Side Menu Bar -> Users -> Create User 
    We will be in user creation page.

    User name: sample_user
    [Tick] Provide user access to the AWS Management Console - optional
    Password: Give special password and remember

    [Untick] Users must create a new password at next sign-in - Recommended

    Give "Next"

    Set permissions
        Permissions Options: (select) Attach policies directly
        [Tick] AmazonS3FullAccess
            ! Provides full access to all buckets via the AWS Management Console.
        [Tick] AmazonS3ReadOnlyAccess
            ! Provides read only access to all buckets via the AWS Management Console.

    Give "Next" 
        First review the page then
    Give "Create User" 

    ! User created successfully

    Now the created user "sample_user" is in Users List page.

    Signout from Root User. 

    Signin To Console. 
        IAM user sign in
            Account ID or alias: 795359014602
            IAM username: sample_user
            Password: <Give Password> 

    !Now logged into IAM User. 



    Check to create s3 bucket and list s3 bucket 
        - Allowed 

        Tasks
            - Try to create bucket 
                It will allow 
                Dont create 
            - Go to List of General Purpoe Buckets 
                It will display the buckets 
                Note: Allowed.
    
    Try other services - to manage. 
        - It will not be allowed. 

        EC2 Instances List - Error we will get.
